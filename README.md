## Building

    $ make build-quick

## Pulling

    $ docker pull graze/php-alpine

## Usage

    $ docker run --rm graze/php-alpine -v $(pwd):/srv php some/script.php

## Running with phpdbg:

    $ docker run --rm graze/php-alpine:test -v $(pwd):/srv phpdbg7 some/script.php
